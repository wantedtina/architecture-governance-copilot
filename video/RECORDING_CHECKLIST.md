# Recording Checklist

## Safety and privacy

- [ ] Clean desktop and browser profile
- [ ] Notifications and Focus interruptions disabled
- [ ] No personal files, usernames, bookmarks, local paths, tokens, or secrets visible
- [ ] No confidential or real internal SI content
- [ ] No real meeting transcript or employee data
- [ ] Synthetic sample confirmed
- [ ] No external service tabs open

## Application state

- [ ] `uv sync` completes
- [ ] `uv run pytest` passes
- [ ] `uv run ruff check .` passes
- [ ] `uv run ruff format --check .` passes
- [ ] Local Streamlit app is healthy
- [ ] `Reset Demo` returns to clean Stage 1
- [ ] `Load Sample Drafting Context` fills all drafting tabs
- [ ] SI generation opens editable draft review
- [ ] SI confirmation routes to Stage 2
- [ ] Deterministic analysis routes to Stage 3
- [ ] Governance confirmation routes to Stage 4
- [ ] Stage 4 stepper shows all stages complete

## Capture setup

- [ ] 1920 × 1080, 30 fps
- [ ] Browser zoom 100%
- [ ] Streamlit sidebar and main content readable
- [ ] Cursor movement deliberate and unhurried
- [ ] No unnecessary scrolling
- [ ] Fixed confirmation button visible
- [ ] Source recording uses only the local app

## Required take

- [ ] Synthetic-data and Domain Architect notice visible
- [ ] Solution Intent and transcript shown
- [ ] `Changes Requested` and all counts shown
- [ ] PostgreSQL decision shown
- [ ] Failover and RTO/RPO findings shown
- [ ] Database-sizing risk shown
- [ ] Both actions shown
- [ ] Evidence quote shown
- [ ] Owner changed to `Taylor Kim`
- [ ] Redis question excluded
- [ ] Generated record reflects Taylor Kim and omits Redis
- [ ] Two Azure DevOps work-item previews shown
- [ ] Preview-only and not-submitted disclosure is visible

## Audio and rehearsal

- [ ] Narration room and microphone checked
- [ ] 10 seconds of room tone recorded
- [ ] Narration pace rehearsed at 125–135 words per minute
- [ ] Two successful end-to-end rehearsals
- [ ] Sentence-level retakes labelled

## Final playback

- [ ] Complete video watched from beginning to end
- [ ] UI remains readable
- [ ] Audio is clear and consistent
- [ ] Subtitles are synchronized and inside safe area
- [ ] Opening and closing identity text is exact
- [ ] Duration is below four minutes
